# Wooden Craft

## Clone this repo
    git clone https://github.com/DeepGajjar31/wooden-craft.git

Open code in VS Code

## Commands run in vs code terminal
    npm install
    npm start

## Use with dockerfile
    # Build docker image
    docker build -t woodencraft .

    # Run docker image
    docker run -itd -p 5000:3000 woodencraft
